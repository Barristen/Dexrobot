# RL-05：基础控制器 + 残差 RL

**类型：** 强化学习 | **触觉支持：** 可选（T04 使用）| **适用任务：** T02, T04, T11

---

## 原始工作

- 论文：[Residual Policy Learning](https://arxiv.org/abs/1812.06298)（Silver et al., 2018）
- 相关工作：[Residual Reinforcement Learning for Robot Control](https://arxiv.org/abs/1812.03201)（Johannink et al., 2019）

---

## 核心思路

**核心假设：** 复杂操控任务可分解为若干简单的运动原语（primitive），每个原语对应一个基础控制器，再用残差 RL 学习在原语基础上的修正策略。

**两层结构：**

| 层次 | 内容 |
|------|------|
| 底层（基础控制器）| 旋转、平移、捏合等运动原语，通过独立 RL 预训练或解析设计 |
| 顶层（残差 RL）| 学习如何修正基础控制器输出以完成高层任务 |

**训练流程：**
1. 预训练各基础原语控制器（独立 RL 任务，简单易收敛）
2. 冻结基础控制器，训练残差网络学习修正量
3. 最终动作 = 基础控制器动作 + 残差输出

**优势：** 降低高层任务的探索难度；尤其适合 T11 长时程串联任务（原语可复用）。
